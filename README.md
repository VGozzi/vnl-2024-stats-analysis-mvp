# vnl-2024-stats-analysis-mvp

Este projeto é um MVP de análise exploratória de dados aplicada ao dataset da VNL Masculino de 2024. O objetivo deste trabalho é investigar se as métricas de desempenho em quadra dos jogadores são capazes de caracterizar e classificar a posição dos atletas, servindo como ponta de partida inicial para a construção de um modelo de classificação supervisionada.

O dataset está disponível publicamente no Kaggle: [VNL 2024 Men's Stats](https://www.kaggle.com/datasets/jonathanpmoyer/vnl-2024-mens-stats)
